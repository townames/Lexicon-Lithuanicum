Lexicon Lithuanicum

This repository preserves a handwritten 17th-century German–Lithuanian dictionary and provides structured data for reuse.

Available files

  • start.html — raw source text and metadata
  • search.html — dedicated bilingual search page
  • dictionary.csv — structured CSV dataset
  • dictionary.json — structured JSON dataset for programmatic use
  • parse_dictionary.py — parser script
